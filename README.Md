## 🌊 About the Game

**RuinSprint** is a 2D endless runner built entirely using **Python and Pygame**.

You play as a fast-moving cat running through mysterious **ancient underwater ruins**, dodging monsters, warriors, and magical enemies while trying to survive as long as possible.

## ✨ Features

> -  Smooth sprite-based animation system
> -  Dynamic scrolling background & ground
> -  `Multiple enemy` types with random behavior
> -  Collision-based game over system
> -  Sound effects + background music
> -  Custom hitbox system for accurate collisions
> -  Interactive menu (Play / Exit / Music toggle)
> -  Score system based on survival time

## 🕹️ Controls

| Key | Action |
